About DevSecure

DevSecure is a cutting-edge website vulnerability scanner built to empower developers, security professionals, and businesses to identify and mitigate security risks. From detecting open ports and missing security headers to uncovering SQL injection and XSS vulnerabilities, DevSecure provides comprehensive analysis with an intuitive interface.

Whether you're securing a personal project, a startup's website, or an enterprise application, DevSecure delivers actionable insights through color-coded results and downloadable PDF reports. Built with modern Python libraries and a Flask backend, it combines performance with ease of use, making cybersecurity accessible to all.

Join a growing community of users who trust DevSecure to safeguard their online presence. Contribute to the project on GitHub or deploy it for your own use in minutes.

Features

URL Scanner

Scan any website for open ports, insecure headers, SQLi/XSS vulnerabilities with a single click.

SSL Certificate Check

Verify SSL certificate validity and ensure secure HTTPS redirection to protect user data.

Whois Lookup

Gather domain and IP information to understand the website's ownership and hosting details.

Exposed Files Detection

Identify sensitive files like .git, .env, or phpinfo() that could expose critical information.

PDF Reports

Generate detailed, downloadable PDF reports summarizing scan results for easy sharing and documentation.
